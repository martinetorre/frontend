proyecto 1 : mi portfolio




El proyecto "mi portfolio" es una aplicación full stack web de arquitectura distribuida, contiene un diseño de interfaz de usuario (front end), una base de datos que almacena la información, que cuenta con las APIs necesarias para proveer a través de internet dicha información (back end).
En ella observarán, la producción hecha con base en los conocimientos alcanzados al momento, colaboración en equipo, datos personales, estudios cursados, algunas de mis experiencias laborales, conocimiento de las tecnologías, dedicación y voluntad e interés de trabajo.
